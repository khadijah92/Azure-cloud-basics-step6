# Azure-cloud-basics-step6
Azure Free tier setup for Cloud course
Project Overview
This project demonstrates setup of Microsoft Azure Free Tier account with cost management, portal navigation, and security best practices for cloud beginners.
Step-by-Step Setup Guide

1. Azure Account Creation
1. Go to https://azure.microsoft.com/free
2. Click "Start free" and sign in with Microsoft account
3. Complete email verification and phone/SMS verification
4. Enter payment method - no charges for Free Tier
5. Verify identity and activate $200 free credit

2. Portal Navigation
1. Login to https://portal.azure.com
2. Explore dashboard: Home → Subscriptions → Resource Groups
3. Use search bar to find Cost Management, Security, Virtual Machines
4. Check subscription status and credit balance

3. Cost Management & Budget Alert
1. Go to Cost Management + Billing
2. Click Budgets → Add new budget
3. Set budget amount $100 and alert threshold at 75%
4. Add email notification for alerts
5. View spending in Cost Analysis dashboard

4. Security Best Practices
1. Go to Microsoft Entra ID → Security
2. Enable Multi-Factor Authentication for account
3. Review Security Center recommendations
4. Understand Shared Responsibility Model

Troubleshooting
Issue: Can't access Free Tier
Fix: Use new Microsoft account, clear browser cache, try incognito mode

Issue:Budget alert emails not received
**Fix:** Check notification email address is correct in budget settings

Issue: MFA setup fails
Fix:Use Microsoft Authenticator app instead of SMS
